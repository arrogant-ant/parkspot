### Environment Setup

```
App will run on  http://parkspot.localhost:8080/. To confifigure the DNS in widows follow below step
Go to C:\Windows\System32\drivers\etc
Open host file
Change "127.0.0.1   localhost" to "127.0.0.1    parkspot.localhost"
```
